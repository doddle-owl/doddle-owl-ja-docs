==================
DODDLE-OWLとは
==================
**DODDLE-OWL** (a Domain Ontology rapiD DeveLopment Environment – OWL extension) は，領域オントロジーを半自動的に構築するためのツールである．DODDLE-OWL はオントロジーにおける階層関係（上位・下位関係）および非階層関係（上位・下位以外の関係）の構築を支援する．入力として，領域における専門文書，または専門文書に含まれる単語集合を与え，汎用オントロジー（EDR　電子化辞書，WordNet，日本語　WordNet, 日本語　Wikipediaオントロジー）と既存　OWL　オントロジーを参照することで，概念階層および非階層関係を半自動的に構築する．また，DODDLE-OWL は，概念階層の洗練を支援するための機能を備えている．最終的に構築した領域オントロジーは，セマンティック Web における標準的なオントロジー記述言語である OWL（Web Ontology Language）形式で出力できる．
